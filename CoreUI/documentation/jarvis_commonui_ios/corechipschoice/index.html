CoreChipsChoice

This is Chips choice button

Declaration

class CoreChipsChoice

Overview

In this height and width will be set automatically according to text.

How to use :

var config = ChipsChoiceConfiguration(isChecked: isChecked, isIcon: isIcon, title: "Label")
config.imageName = .ic_camera
button = ChipsChoice(frame: frame)
button?.delegate = self
button?.setConfig(config)
