How to USE:

headerView = CoreHeaderDefault(frame: CGRect(x: 0, y: statusBarHeight, width: self.view.frame.width, height: 0))
headerView.setConfig(CoreHeaderLogoConfiguration(actionIcon: actionic, isDark: isDarkk, icon: actionic == .Yes ? (image) : (nil),headerLogoImage: .paytm))

``` OR (For custom back or close button usage) write below code

headerView.setConfig(CoreHeaderLogoConfiguration(actionIcon: actionic, isDark: isDarkk, icon: actionic == .Yes ? (image) : (nil),headerLogoImage: .paytm,headButtonImageName: .close))
headerView.delegate = self

Height is dynamic and Width is given by verticle
