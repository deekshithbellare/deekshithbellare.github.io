Overview

How to USE:

headerView = CoreHeaderDefault(frame: CGRect(x: 0, y: 0, width: self.view.frame.width, height: 104))
headerView.setConfig(CoreHeaderDefaultConfiguration(isExtended: .Yes, title: .Yes, subtitle: .Yes, iconCount: .None, icons: [.add, .camera], isDark: .Yes))
headerView.set(title: "Title", subTitle: "Subtitle")
headerView.delegate = self

Height is dynamic and Width is given by verticle
