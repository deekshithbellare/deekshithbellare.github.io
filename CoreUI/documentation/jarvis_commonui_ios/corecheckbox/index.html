CoreCheckbox

This is the checkbox button

Declaration

class CoreCheckbox

Overview

How to Use:

let config = CheckboxConfiguration(checkboxState: stateVal, checkboxLabel: labelVal, checkboxEnabled: enabledVal)
checkboxBtn = CoreCheckbox(frame: CGRect(x: 0, y: 400, width: baseview.frame.width, height: 50))
checkboxBtn.setupConfiguration(config, title: "label")
checkboxBtn.delegate = self

Get current state of checkbox button

checkboxBtn.isSelected()
