CoreFabButton

UIButton subclass for showing Fab Button.

Declaration

class CoreFabButton

Overview

Uses

  1. With StoryBoard/Xib

    drag a UIButton and set the class as CoreFabButton and set IBOutlet. After setting outlet , set the configuration

    let config = CoreFabButtonConfig(title: "Storyboard Button", icon: UIImage(named: "icon_rg24_system_bounding")!)
let fabButtonStoryboard?.config = config
fabButtonStoryboard?.callback = {
}

  2. With Code.

    let config = CoreFabButtonConfig(title: title, icon: UIImage(named: "icon_rg24_system_bounding")!)
self.fabButton = CoreFabButton(config: config, callback: {
})
