Protocol

CoreBottomSheetPresentable

This is the configuration object for a view controller that will be presented using the CoreBottomSheet transition.

Declaration

protocol CoreBottomSheetPresentable : AnyObject

Overview

Usage:

extension YourViewController: CoreBottomSheetPresentable {
func shouldRoundTopCorners: Bool { return false }
}

Topics

Instance Properties

var allowsDragToDismiss: Bool
A flag to determine if dismissal should be initiated when swiping down on the presented view.
var allowsExtendedPanScrolling: Bool
A flag to determine if scrolling should seamlessly transition from the bottomsheet container view to the embedded scroll view once the scroll limit has been reached.
var allowsTapToDismiss: Bool
A flag to determine if dismissal should be initiated when tapping on the dimmed background view.
var anchorModalToLongForm: Bool
A flag to determine if scrolling should be limited to the longFormHeight. Return false to cap scrolling at .max height.
var bottomSheetBackgroundColor: UIColor
The background view color.
var bottomSheetScrollView: UIScrollView?
The scroll view embedded in the view controller. Setting this value allows for seamless transition scrolling between the embedded scroll view and the bottom sheet container view.
var cornerRadius: CGFloat
The corner radius used when shouldRoundTopCorners is enabled.
var dragIndicatorBackgroundColor: UIColor
The drag indicator view color.
var isHapticFeedbackEnabled: Bool
A flag to determine if haptic feedback should be enabled during presentation.
var isUserInteractionEnabled: Bool
A flag to toggle user interactions on the container view.
var longFormHeight: CoreBottomSheetHeight
The height of the bottomsheet container view when in the longForm presentation state.
var scrollIndicatorInsets: UIEdgeInsets
We configure the bottomSheetScrollView’s scrollIndicatorInsets interally so override this value to set custom insets.
var shortFormHeight: CoreBottomSheetHeight
The height of the bottomsheet container view when in the shortForm presentation state.
var shouldRoundTopCorners: Bool
A flag to determine if the top corners should be rounded.
var showDragIndicator: Bool
A flag to determine if a drag indicator should be shown above the bottomsheet container view.
var springDamping: CGFloat
The springDamping value used to determine the amount of ‘bounce’ seen when transitioning to short/long form.
var topOffset: CGFloat
The offset between the top of the screen and the top of the bottomsheet container view.
var transitionAnimationOptions: UIView.AnimationOptions
The animation options used when performing animations on the CoreBottomSheet, utilized mostly during a transition.
var transitionDuration: Double
The transitionDuration value is used to set the speed of animation during a transition, including initial presentation.

Instance Methods

func bottomSheetAnimate(Self.AnimationBlockType, Self.AnimationCompletionType?)
A function wrapper over the animate function in CoreBottomSheetAnimator.
func bottomSheetDidDismiss()
Notifies the delegate after the bottomsheet is dismissed.
func bottomSheetPerformUpdates(() -> Void)
Operations on the scroll view, such as content height changes, or when inserting/deleting rows can cause the bottomsheet to jump, caused by the bottomsheet responding to content offset changes.
func bottomSheetSetNeedsLayoutUpdate()
A function wrapper over the setNeedsLayoutUpdate() function in the CoreBottomSheetPresentationController.
func bottomSheetTransition(to: CoreBottomSheetPresentationController.PresentationState)
A function wrapper over the transition(to state: CoreBottomSheetPresentationController.PresentationState) function in the CoreBottomSheetPresentationController.
func bottomSheetWillDismiss()
Notifies the delegate that the bottomsheet is about to be dismissed.
func shouldPrioritize(CoreBottomSheetGestureRecognizer: UIPanGestureRecognizer) -> Bool
Asks the delegate if the bottomsheet gesture recognizer should be prioritized.
func shouldRespond(to: UIPanGestureRecognizer) -> Bool
Asks the delegate if the bottomsheet should respond to the bottomsheet gesture recognizer.
func shouldTransition(to: CoreBottomSheetPresentationController.PresentationState) -> Bool
Asks the delegate if the bottomsheet should transition to a new state.
func willRespond(to: UIPanGestureRecognizer)
Notifies the delegate when the bottomsheet gesture recognizer state is either began or changed. This method gives the delegate a chance to prepare for the gesture recognizer state change.
func willTransition(to: CoreBottomSheetPresentationController.PresentationState)
Notifies the delegate that the bottomsheet is about to transition to a new state.
